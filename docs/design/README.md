## 🎯 设计原则

1. **静态分析优先** - 不依赖代码运行，纯静态分析
2. **知识驱动** - 利用领域知识增强分析能力
3. **跨平台** - 支持 Windows/Linux/macOS
4. **可扩展** - 模块化设计，易于扩展新语言和平台
